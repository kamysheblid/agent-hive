# v1.7.2

## Summary

**Massive install speed optimization!** Reduced from **384 packages → ~10 packages**.

## Changes

### Breaking Change: All Optional Packages Moved to peerDependenciesMeta

Previously, optional tools (agent-booster, memory, ast-grep, MCPs) were installed automatically with `npm install`. This caused:
- Slow installation (~47 seconds)
- Large dependency tree (384 packages)
- tree-sitter compilation issues

**Now:** Plugin installs only core dependencies. Optional tools are **not installed automatically** - user installs them manually if needed.

## What Changed

| Before | After |
|--------|-------|
| 384 packages installed | ~10 packages installed |
| ~47s install time | ~2s install time |
| tree-sitter issues | No native deps by default |

## Optional Tools (Install Manually)

```bash
# Fast code editing (52x faster)
npm install @sparkleideas/agent-booster

# Vector memory
npm install @sparkleideas/memory

# AST code analysis (needs C++20 on Node v24+)
npm install @ast-grep/napi

# Code search
npm install grep-mcp

# Documentation lookup
npm install @upstash/context7-mcp

# Web search
npm install exa-mcp-server

# Code navigation
npx -y @butttons/dora

# Auto code review
npx -y auto-cr-cmd
```

## Doctor Still Works

Run `hive-doctor` to see what tools are missing and get install commands:

```bash
npx @hung319/opencode-hive doctor
```

## Installation

```bash
# Core only (fast)
npm install @hung319/opencode-hive@1.7.2

# With recommended tools
npm install @hung319/opencode-hive && npm install @sparkleideas/agent-booster grep-mcp
```
