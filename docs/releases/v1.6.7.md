# v1.6.7

## Breaking Fix

### @ast-grep/napi is now truly optional

**Problem:** Plugin failed to load when `@ast-grep/napi` native build failed (tree-sitter needs C++20 on Node 24+).

**Solution:** 
- Moved `@ast-grep/napi` from `dependencies` to `optionalDependencies`
- Plugin now loads successfully even if native build fails
- If `@ast-grep/napi` is not available, functionality degrades gracefully

**Note:** For full ast-grep functionality, install manually:
```bash
export CXXFLAGS="-std=c++20"
npm install @ast-grep/napi
```
