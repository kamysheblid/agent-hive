# v1.10.3

## Bug Fixes

- **Fix --install path parsing** — Fixed issue where `doctor --install /path` incorrectly parsed path as `/root/doctor` instead of the provided path

### Root Cause

When running `npx @hung319/opencode-hive doctor --install /root/.local`, the argument parser was finding the first non-flag argument incorrectly. The fix now properly looks for the path argument immediately after the `--install` or `-i` flag.

### Before (broken)
```bash
$ npx @hung319/opencode-hive doctor --install /root/.local
🔍 Scanning: /root/doctor  # ❌ Wrong path!
```

### After (fixed)
```bash
$ npx @hung319/opencode-hive doctor --install /root/.local
🔍 Scanning: /root/.local  # ✅ Correct path!
```
