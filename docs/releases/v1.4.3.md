# Release v1.4.3

## New Features

### Agent Booster Tools 🔥

Ultra-fast code editing powered by Rust+WASM.

| Tool | Description |
|------|-------------|
| `hive_code_edit` | 52x faster code editing vs Morph LLM |
| `hive_lazy_edit` | Edit with `// ... existing code ...` markers |
| `hive_booster_status` | Check availability |

**Features:**
- 52x faster than Morph LLM
- FREE - No API key required
- Rust+WASM powered
- Graceful fallback to native edit

### Vector Memory Tools 🔍

Semantic memory search with HNSW indexing.

| Tool | Description |
|------|-------------|
| `hive_vector_search` | Find memories by meaning |
| `hive_vector_add` | Add memories with embeddings |
| `hive_vector_status` | Check vector index status |

**Features:**
- HNSW indexing for fast similarity search
- Hybrid search (vector + keyword)
- Filter by type and scope
- Automatic fallback to text search
