# v1.4.4 Release Notes

**Date**: 2026-03-20

## Highlights

This release introduces native ast-grep tools, @paretools/search integration, and a new health check system.

## New Features

### Native ast-grep Tools

Replaced MCP-based ast-grep with native `@ast-grep/napi` for **faster performance** and **no external dependencies**.

| Tool | Description |
|------|-------------|
| `ast_grep_dump_syntax_tree` | Dump code's syntax structure |
| `ast_grep_test_match_code_rule` | Test code against YAML rules |
| `ast_grep_find_code` | Find code matching patterns |
| `ast_grep_scan_code` | Scan for TypeScript bugs |
| `ast_grep_rewrite_code` | Transform/refactor code |
| `ast_grep_analyze_imports` | Analyze import usage |

### @paretools/search Integration

New MCP server `pare_search` for structured ripgrep/fd output with **65-95% token reduction**.

```json
{
  "mcpServers": {
    "pare_search": {
      "command": "npx",
      "args": ["-y", "@paretools/search"]
    }
  }
}
```

### Hive Doctor

New health check tool to diagnose system issues and get optimization recommendations.

```typescript
// Full health check
hive_doctor()

// Quick status
hive_doctor_quick()
```

**Checks:**
- Dependencies installed
- Optimizations enabled
- Configuration issues
- Recommendations with quick fixes
