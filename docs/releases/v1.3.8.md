# Release v1.3.8

## Bug Fixes

### Agent Configuration
- **Primary Agent**: `zetta` is now the primary agent
- **Subagent Mode**: OpenCode's built-in agents (`build`, `plan`, `triage`, `docs`, etc.) are demoted to `mode: 'subagent'`
- **Default Agent**: `default_agent` is explicitly set to ensure `zetta` is used as primary

This ensures that when users launch OpenCode with the @hung319/opencode-hive plugin, the `zetta` agent (hybrid planner + orchestrator) becomes the default, not OpenCode's default `build` or `plan` agents.
