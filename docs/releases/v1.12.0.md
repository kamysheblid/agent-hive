# v1.12.0

## 5 Community Plugin Patterns

Adopted 5 community-driven enhancement patterns into the OpenCode Hive plugin — compaction restoration, hidden-session judge, OpenCode provider auto-capture, user profile learning, and zero-risk principle.

### New Features

- **🔄 Compaction Restoration** — Re-injects vector memories via `session.prompt` after session compact. No memory loss on compact cycles. Configurable via `compactionRestoration.enabled` in `agent_hive.json`.

- **🔍 Hidden-Session Judge** — 5 heuristic detectors (timely, accurate, coherent, relevant, complete) + LLM-powered judge prompt for evaluating task completion quality post-execution. No visible output — silent scoring. Configurable via `hiddenJudge.enabled`.

- **📸 OpenCode Provider Auto-Capture** — Piggybacks on `session.prompt` to capture structured memory snapshots automatically via `generateStructured()`. No extra API keys — works with any OpenCode provider. Configurable via `autoCapture.enabled`.

- **🧠 User Profile Learning** — Periodic AI analysis of conversation patterns with confidence decay and persistence. Builds a preference profile over time (communication style, expertise level, preferences, anti-preferences). Injected into system prompt via `system.transform` hook. Configurable via `userProfile.enabled`.

- **🛡️ Zero-Risk Principle** — `safeStage`/`safeStageAsync`/`safeHook`/`conservativeFilter` wrappers across all 6 hook handlers and all 7 VectorMemoryService public methods. Best-effort, never crashes the plugin. Conservative filter: string output > 3x fallback length triggers fallback.

### Configuration

All features opt-in via `~/.config/opencode/agent_hive.json`:

```json
{
  "plugins": {
    "opencode-hive": {
      "compactionRestoration": { "enabled": true },
      "hiddenJudge": { "enabled": false },
      "autoCapture": { "enabled": true },
      "userProfile": { "enabled": false }
    }
  }
}
```

- `utils/compaction-restoration.ts` — `restoreMemories()` using `searchMemories('', { limit })` two-arg signature

- `services/hidden-judge.ts` — `HiddenJudgeService` with semantic prompt injection via `session.prompt`
- `services/opencode-provider.ts` — `OpenCodeProviderService.generateStructured()` via `session.prompt`
- `utils/profile-prompt.ts` — `buildProfileAnalysisPrompt()`, `parseProfileAnalysisResponse()`
- `services/user-profile.ts` — `UserProfileService` with periodic analysis, confidence decay, persistence
- `utils/safe-stage.ts` — `safeStage`/`safeStageAsync`/`safeHook`/`conservativeFilter`
- `services/vector-memory.ts` — All 7 public methods wrapped with `safeStageAsync`
- `index.ts` — All 6 hooks wrapped with `safeHook`, user message tracking, profile injection
