# v1.5.8

## What's Changed

### Enhanced: `hive_doctor` with actionable output

Rewrote the doctor tool to provide clear, actionable guidance for both AI agents and humans.

**New output structure:**
- `summary`: Human-readable status per category
- `actionItems[]`: Prioritized fixes (high/medium/low)
- `quickInstall`: Ready-to-run install commands

**Checks expanded:**
- Dependencies: 8 packages checked
- CLI Tools: dora, auto-cr, scip-typescript, veil
- Native Binaries: @ast-grep/napi tree-sitter status
- Config: 7 optimization features

**Setup workflow guidance:**
- For LLM: Parse actionItems → install → verify → done
- For Human: Run doctor → see summary → install what you need

### Documentation: Quick Setup section

Added "Quick Setup (3 Steps)" section at the top of README:
- For AI Agents: 3-line setup workflow
- For Humans: Step-by-step with copy-paste commands
