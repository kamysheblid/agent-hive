# Release v1.3.7

## Features

### LSP Tools (IDE-like Functionality)
| Tool | Description |
|------|-------------|
| `lsp_rename` | Rename symbol across files |
| `lsp_goto_definition` | Jump to symbol definition |
| `lsp_find_references` | Find all references to a symbol |
| `lsp_diagnostics` | Get file diagnostics (errors, warnings) |
| `lsp_hover` | Get type info and documentation |
| `lsp_code_actions` | Get available code actions |

### Skill-Embedded MCPs
- Skills can now bring their own MCP servers
- Configure MCP in skill SKILL.md frontmatter:
```yaml
---
mcp:
  playwright:
    command: npx
    args: ["-y", "@anthropic-ai/mcp-playwright"]
---
```
- Use `skill_mcp` tool to invoke skill MCPs
- Use `list_skill_mcps` tool to see available MCPs
