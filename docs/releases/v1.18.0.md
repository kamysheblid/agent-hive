# v1.18.0

## Features

### Code Search (BM25 + AST + Symbol Fusion)
- **`code_search`** — Unified search combining BM25 keyword, AST pattern, and SCIP symbol search
- **`code_search_index`** — Build/rebuild BM25 index on-demand
- **`code_search_status`** — Index stats (files indexed, term count)
- Pure TypeScript BM25 implementation, no external dependencies
- Symbol search via dora CLI (optional, graceful fallback)

### Call Graph (Tree-sitter)
- **`call_graph_extract`** — Extract all calls from a single file
- **`call_graph_callees`** — Find all functions called by a given function
- **`call_graph_callers`** — Find all functions that call a given function
- **`call_graph_path`** — Find shortest call path between two functions (BFS)
- Regex-based extraction, no external binary required

### LSP Phase 1 — Fixed Auto-Install
- Fixed EPERM issue by switching to local prefix install (`~/.config/opencode/hive/lsp`)
- Added Python diagnostics via pyright
- Updated LSP_DATABASE with local install paths

### LSP Phase 2 — Real LSP Client
- JSON-RPC 2.0 transport over stdio
- LSP Client supporting: initialize, openFile, closeFile, gotoDefinition, findReferences, hover, rename, diagnostics
- Client lifecycle management with idle reaper (5 min timeout)
- All 8 `lsp_*` tools now use real LSP instead of stubs

### Project Memory Improvements
- Auto-save project memory on task completion (`hive_worktree_commit` status=done)
- Deduplication: same feature + same day → overwrite (no spam)
- Agent instruction: update project memory after completing work

### Gitignore Automation
- Auto-add `.hive/` and `.hive2/` to `.gitignore` on plugin init
- Safe to call multiple times — no-ops if already present
