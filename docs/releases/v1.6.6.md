# v1.6.6

## Improvements

### Better CXXFLAGS detection

Doctor now shows clear status about CXXFLAGS:

| Status | Meaning | Action |
|--------|---------|--------|
| `✓ Active in session` | CXXFLAGS is set and active | Ready! |
| `⚠ Configured but not active` | In ~/.bashrc but not sourced | Run: `source ~/.bashrc` |
| `○ Not set` | Not configured | Run with `--fix` |

### Auto-fix mode improvements

- `--fix` sets CXXFLAGS for current session
- Adds to ~/.bashrc for future sessions
- Shows hint about sourcing bashrc

### CI tests added

- Test standalone doctor runs without errors
- Test `--fix` mode works
