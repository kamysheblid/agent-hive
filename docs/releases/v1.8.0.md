# v1.8.0

## Summary

Plugin optimization with new search MCPs, lazy loading, and improved doctor tool.

## New Features

### 1. New Search MCPs

Added two new free search options:

| MCP | Description | API Key |
|-----|-------------|---------|
| **ddg_search** | DuckDuckGo search | None (free) |
| **searxng** | Privacy meta-search | Optional (self-hostable) |

Install:
```bash
npm install @oevortex/ddg_search  # DuckDuckGo
npm install mcp-searxng           # SearXNG
```

### 2. Lazy Load MCP Initialization

MCPs are now lazily initialized - they're only resolved when first accessed, making plugin load faster.

### 3. Doctor First-Run Detection

First time running doctor? You'll see a friendly welcome message.

### 4. MCP Guide Updated

- Added documentation for ddg_search and searxng
- Removed outdated veil references
- Updated search selection guide

## Changes

| Before | After |
|--------|-------|
| 4 search MCPs | 6 search MCPs |
| Eager MCP loading | Lazy MCP loading |
| No first-run message | Friendly first-run message |

## Installation

```bash
npm install @hung319/opencode-hive@1.8.0
```

## Optional: Install New MCPs

```bash
# Free DuckDuckGo search
npm install @oevortex/ddg_search

# Privacy meta-search  
npm install mcp-searxng

# Or use --install to auto-install all
npx @hung319/opencode-hive doctor --install
```
