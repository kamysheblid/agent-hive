# v1.5.7

## What's Changed

### Fix: Make ast-grep tools truly optional with graceful fallback

Previously, `@ast-grep/napi` native binaries required tree-sitter compilation which could fail on some environments (Node.js v24, certain CI configurations).

**New dual-mode approach:**

1. **Native mode** (fastest): Uses `@ast-grep/napi` if native binaries exist
2. **CLI mode** (works everywhere): Falls back gracefully to MCP via npx

**How it works:**

- Check for `.node` native binary files **before** importing
- If native binaries exist → try to load `@ast-grep/napi`
- If native load fails → fall back to CLI mode
- CLI mode suggests using the MCP tool for full functionality

**Benefits:**

- Plugin works even without native binary compilation
- Tools never block or crash on missing dependencies
- Clear feedback about which mode is active
- MCP ast_grep tool remains available for all users
