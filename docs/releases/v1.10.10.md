# v1.10.10

## Features

- **Auto-symlink binaries** — After installing CLI tools, automatically create symlinks in `bin` folder

### How It Works

When installing CLI tools to a custom prefix:
1. Install package to `targetPath/node_modules`
2. Create `targetPath/bin` directory if needed
3. Symlink binaries from `node_modules/.bin` to `bin/`

### Example
```bash
npx @hung319/opencode-hive doctor --install /root/.local
```

Output:
```
  Installing CLI searxng...
    ✓ searxng ready (symlinked)
```

Now binaries are available at `/root/.local/bin/` which is added to PATH.
