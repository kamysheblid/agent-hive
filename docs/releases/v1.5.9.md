# v1.5.9

## What's Changed

### NEW: Standalone Doctor - Run BEFORE installing!

Check if your system is ready **before** installing the plugin:

```bash
bunx @hung319/opencode-hive doctor
npx @hung319/opencode-hive doctor
```

This shows:
- Missing dependencies with install commands
- CLI tools status
- Native binary availability
- Config file status

### Enhanced Doctor Checks

**More dependencies checked:**
- Core: `@ast-grep/napi`, `@notprolands/ast-grep-mcp`, `@paretools/search`
- Agent: `@sparkleideas/agent-booster`, `@sparkleideas/memory`
- MCPs: `@upstash/context7-mcp`, `exa-mcp-server`, `grep-mcp`
- Blockchain: `btca-ask`, `opencode-model-selector`

**More CLI tools checked:**
- `dora`, `auto-cr`, `scip-typescript`, `veil`, `btca`, `ast-grep`

**Human-friendly output:**
- Color-coded status
- Priority levels (critical/high/medium/low)
- Ready-to-copy install commands
- Clear summary of what's missing

### Setup Workflow

**For Humans:**
1. `bunx @hung319/opencode-hive doctor` - Check system first
2. `npm install @hung319/opencode-hive` - Install plugin
3. Install extras you need - Commands shown in doctor output
