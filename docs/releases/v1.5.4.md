# v1.5.4 Release Notes

**Date**: 2026-03-20

## Highlights

This release simplifies the plugin by removing optional dependencies that cause installation issues, while keeping full functionality through graceful fallbacks.

## Changes

### Removed

- **Removed `@ast-grep/napi`** from optionalDependencies
  - Native NAPI compilation fails on Node.js v24 due to tree-sitter C++20 requirements
  - MCP-based ast-grep (`@notprolands/ast-grep-mcp`) continues to work as fallback

- **Removed `autoInstallDeps`** feature
  - npm global install causes conflicts (minipass PathAlreadyExists)
  - Feature was opt-in and rarely used

### How ast-grep Works Now

1. **Primary**: MCP-based ast-grep via `@notprolands/ast-grep-mcp`
   - Available as `ast_grep` MCP server
   - Works out of the box

2. **Optional Enhancement**: Native ast-grep via `@ast-grep/napi`
   - Install manually if needed: `npm install @ast-grep/napi`
   - 52x faster than MCP-based
   - Requires compatible Node.js version (< v24 or with C++20 support)

### Tools Available

All tools continue to work:

| Category | Tools |
|----------|-------|
| Hive | Feature, Plan, Task, Worktree, Context tools |
| MCP | websearch, context7, grep_app, ast_grep, pare_search, veil |

## Migration

### For Users Who Need Native ast-grep

If you need the faster native ast-grep:

```bash
# Install manually in a separate location
npm install @ast-grep/napi

# Or use MCP-based version (works automatically)
# No action needed
```

### If Plugin Fails to Load

1. Clear npm cache: `npm cache clean --force`
2. Remove plugin: `opencode plugin remove @hung319/opencode-hive`
3. Re-add: `opencode plugin add @hung319/opencode-hive`
