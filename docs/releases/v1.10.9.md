# v1.10.9

## Bug Fixes

- **Dynamic PATH** — Use `targetPath/bin` instead of hardcoded `/root/.local/bin`

### Before (hardcoded)
```typescript
const pathEntry = 'export PATH="/root/.local/bin:$PATH"';
```

### After (dynamic)
```typescript
const targetBin = path.join(targetPath, 'bin');
const pathEntry = `export PATH="${targetBin}:$PATH"`;
```

Now works with any prefix:
```bash
npx @hung319/opencode-hive doctor --install /usr/local
# Adds export PATH="/usr/local/bin:$PATH"
```
