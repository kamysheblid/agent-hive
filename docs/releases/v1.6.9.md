# v1.6.9

## Summary

Improved graceful degradation for bundled tools by externalizing all native dependencies at build time, ensuring the plugin loads even if optional native modules fail to compile.

## Changes

### Build Configuration
- **Externalized packages**: `@ast-grep/napi`, `@sparkleideas/agent-booster`, `@sparkleideas/memory`, and `simple-git` are now externalized in the bundle
- Plugin loads gracefully when these packages fail to compile or load

### Doctor Tool Improvements
- **Loadability checks**: Doctor now tests if packages can actually be `require()`d, not just if they're installed
- Detects "installed but broken" state (native build failed, missing platform binary)
- Added `@ast-grep/napi` to agent tools check

### Documentation
- Updated README to list all 3 bundled tools requiring C++20 on Node.js v24

## Installation

```bash
npm install @hung319/opencode-hive@1.6.9
```

For Node.js v24+:
```bash
export CXXFLAGS="-std=c++20"
npm install @hung319/opencode-hive@1.6.9
```

## Upgrade Path

No breaking changes. Upgrade from v1.6.8 or earlier.
