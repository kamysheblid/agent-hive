# v1.9.0

## Summary

Major upgrade with Smart Context Engine and Performance optimizations.

## New Features

### 1. Pattern Learning System

Automatically learns from task execution to improve future suggestions.

```typescript
import { getPatternLearner } from '@hung319/opencode-hive';

// Learn from task
learner.learn('feature', 'add_tests', success, context);

// Get predictions
const suggestions = learner.predict('working_on_api');
```

**Storage:** `.hive/patterns.json`

### 2. Auto-Summary Utility

Automatically summarize task completion and suggest next actions.

```typescript
import { extractKeyChanges, suggestNextTasks } from '@hung319/opencode-hive';

const summary = extractKeyChanges(gitDiff);
const nextActions = suggestNextTasks(['completed', 'tasks'], projectContext);
```

### 3. Incremental Loader

LRU cache for efficient resource loading.

```typescript
import { IncrementalLoader, SkillLoader } from '@hung319/opencode-hive';

const loader = new SkillLoader();
const template = await loader.get('brainstorming');
```

### 4. Background Sync

Non-blocking background synchronization.

```typescript
import { getBackgroundSync } from '@hung319/opencode-hive';

const sync = getBackgroundSync({ interval: 30_000 });
sync.start();
```

## Performance Improvements

| Component | Before | After |
|-----------|--------|-------|
| Skills loading | Full load | Incremental (LRU) |
| MCP loading | Lazy init | Cached + TTL |
| Pattern storage | N/A | Auto-save |
