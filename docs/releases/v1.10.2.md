# v1.10.2

## Bug Fixes

- **Version bump** — Fixed hardcoded version in doctor.ts files showing wrong version
- **README updates** — Updated all READMEs with latest MCPs and environment variables

### README Improvements

- **Added Environment Variables section** — Clear documentation of required env vars:
  - `EXA_API_KEY` — For Exa AI web search
  - `SEARXNG_URL` — For SearXNG meta-search  
  - `CXXFLAGS="-std=c++20"` — For Node.js v24+ native modules
- **Updated MCP tables** — Added `ddg_search` and `searxng` to available MCPs
- **Removed deprecated** — Removed `veil` (conflicts with tree-sitter-typescript)

### MCP List

| MCP | Description | Requirements |
|-----|-------------|--------------|
| `websearch` | Exa AI web search | `EXA_API_KEY` |
| `context7` | Library docs lookup | None |
| `grep_app` | GitHub code search | None |
| `ddg_search` | DuckDuckGo search | None (free) |
| `searxng` | Privacy meta-search | `SEARXNG_URL` |
| `ast_grep` | AST code analysis | None |
