# v1.7.0

## Summary

Removed `veil` and `scip-typescript` from the plugin due to installation and runtime issues.

## Breaking Changes

- **Removed `veil` MCP** - Code discovery now uses `grep-mcp` and `ast_grep` instead
- **Removed `scip-typescript` CLI tool** - TypeScript indexing via CLI no longer available

## Why

### Veil Issues
- Requires manual `veil init` (interactive setup)
- Causes MCP timeouts (30s) when not initialized
- Functionality covered by `grep-mcp` and `ast_grep`

### scip-typescript Issues
- Tree-sitter version conflicts with `@ushiradineth/veil`
- Native compilation failures on some platforms
- Functionality not core to the plugin

## Alternative Tools

| Task | Use Instead |
|------|-------------|
| Code discovery | `grep-mcp`, `ast_grep` |
| Pattern search | `ast_grep_rewrite_code`, `grep_app` |
| File search | `pare_search` |

## Installation

```bash
npm install @hung319/opencode-hive@1.7.0
```
