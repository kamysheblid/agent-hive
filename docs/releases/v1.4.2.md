# Release v1.4.2

## New Features

### Snip Integration
- **Token Reduction**: Shell commands prefixed with `snip` CLI for 60-90% token savings
- **Auto-prefix**: Commands automatically wrapped when `snip` is installed
- **Config**: Enable via `snip.enabled` in `agent_hive.json`

### Enhanced Memory Tools
- **Recall**: `hive_memory_recall` - Search and retrieve typed memories
- **Update**: `hive_memory_update` - Modify existing memory entries
- **Forget**: `hive_memory_forget` - Remove specific memories

### Smart Session Titles
- **Auto-generation**: Session titles based on conversation context
- **Event-driven**: Hooks into session creation events
- **Config**: Enable via `smartTitle.enabled` in `agent_hive.json`

## Configuration Options

```json
{
  "snip": {
    "enabled": true
  },
  "smartTitle": {
    "enabled": true
  }
}
```
