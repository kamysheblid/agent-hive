# v1.14.1

## Features

### LSP Auto-Install

- **New service**: `lsp-autoinstall.ts` — tự động cài language servers khi plugin init
- TypeScript: `typescript-language-server` (npm)
- Python: `pyright` (uv/pip)
- Go: `gopls` (go install)
- Rust: `rust-analyzer` (rustup)
- Fire-and-forget: không block startup, skip nếu đã installed

### Cleanup

- **Removed OpenSERP**: Xoá toàn bộ code, MCP config, docs liên quan tới OpenSERP
- **Removed dead `mcp` property** từ top-level hooks (OpenCode ignore)

## Breaking Changes

- `openserp_search`, `openserp_mega_search`, `openserp_image_search`, `openserp_extract` tools không còn available
- Plugin init sẽ attempt cài language servers (có thể consume disk/network lần đầu)
