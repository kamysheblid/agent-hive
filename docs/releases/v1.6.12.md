# v1.6.12

## Summary

Moved `@ast-grep/napi` from `dependencies` to `optionalDependencies` to fix installation failures on platforms where tree-sitter native compilation fails.

## Changes

- **@ast-grep/napi is now optional** - Plugin installs even if tree-sitter compilation fails
- Updated README to clarify that bundled tools are optional

## Why This Fix

Previously, `@ast-grep/napi` in `dependencies` caused `npm install` to fail entirely if tree-sitter couldn't compile. Now:
- Plugin installs successfully
- ast-grep features work if the native module installed
- Plugin works fine without ast-grep

## Installation

```bash
npm install @hung319/opencode-hive@1.6.12
```

## Optional: Install ast-grep manually

If you want ast-grep code analysis:
```bash
export CXXFLAGS="-std=c++20"
npm install @ast-grep/napi
```
