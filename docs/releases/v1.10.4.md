# v1.10.4

## Features

- **Auto-install mode** — `doctor --install` now automatically installs missing tools instead of just scanning

### How it works

When you run `npx @hung319/opencode-hive doctor --install`:

1. Runs doctor check to find what's missing
2. Auto-installs missing CLI tools via `npx -y`
3. Auto-installs missing agent tools via `npm install`
4. Verifies installation by running doctor again

### Before (v1.10.3)
```bash
$ npx @hung319/opencode-hive doctor --install /root/.local
🔍 Scanning: /root/.local
Found 1 package.json(s)
⚠ No optional dependencies found  # ❌ Just scanned!
```

### After (v1.10.4)
```bash
$ npx @hung319/opencode-hive doctor --install
🔧 Auto-installing missing tools...
  Installing CLI ddg_search...
    ✓ ddg_search installed
  Installing agent tool @sparkleideas/agent-booster...
    ✓ @sparkleideas/agent-booster installed
✓ Auto-install complete
```
