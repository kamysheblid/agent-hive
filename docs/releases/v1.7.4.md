# v1.7.4

## Summary

New `--install` flag to auto-install optional dependencies from `peerDependenciesMeta`.

## Changes

### New: `--install` flag

Scan and install optional dependencies from `peerDependenciesMeta`:

```bash
# Install in current directory
bunx @hung319/opencode-hive doctor --install

# Install in specific project
bunx @hung319/opencode-hive doctor --install /path/to/project
bunx @hung319/opencode-hive doctor --install /root/.local/share/opencode
```

### What it does

1. Scans for `package.json` files (up to 3 levels deep)
2. Finds optional dependencies from `peerDependenciesMeta`
3. Installs missing ones with `npm install`
