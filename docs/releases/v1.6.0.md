# v1.6.0

## Breaking Changes

### Removed: ast-grep MCP and native tools

**Ast-grep is no longer bundled with this plugin.** If you need ast-grep functionality:

1. Install separately: `npm install @ast-grep/napi` or use `@notprolands/ast-grep-mcp`
2. Configure it separately in your `.opencode/mcp-servers.json`

This removes native binary dependencies that caused build issues on Node.js v24.

### Updated: btca package

- Changed from `btca-ask` to `btca`
- Install: `npm install btca`

---

## What's Changed

### Doctor checks simplified

- Removed native binary checks
- Updated btca to `btca` (from npmx.dev/package/btca)
- Cleaner dependency list

### Cleanup

- Removed `@ast-grep/napi` dependency
- Removed `--external @ast-grep/napi` from build
- Removed `bunfig.toml` trustedDependencies
- Removed CI `BUN_INSTALL_ALLOW_SCRIPTS` env var
