# v1.10.8

## Bug Fixes

- **Syntax error fix** — Removed duplicate return statement causing parse error
- **Auto-add PATH** — After installing CLI tools, automatically adds `/root/.local/bin` to shell PATH

### New Feature: Auto PATH Configuration

After `--install`, doctor now:
1. Installs packages to target path
2. Detects shell config files (`.bashrc`, `.bash_profile`, `.zshrc`)
3. Adds `export PATH="/root/.local/bin:$PATH"` if not already present
4. Shows hint to source shell config

### Usage
```bash
npx @hung319/opencode-hive doctor --install /root/.local
# Installs tools and adds to PATH automatically
```

### Output Example
```
🔧 Auto-installing to: /root/.local
  Installing CLI searxng...
    ✓ searxng ready
    ✓ Added PATH to /root/.bashrc
    ✓ Added PATH to /root/.bash_profile

💡 Run this to use new tools in current session:
  source ~/.bashrc  # or ~/.zshrc
```
