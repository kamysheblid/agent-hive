# v1.10.1

## What's New

This release updates the README documentation to reflect v1.10 features and bumps the version.

### Documentation Updates

- **README redesign** — New "What's New in v1.10" section highlighting key features
- **Feature overview tables** — MCP servers, tools, and utilities organized by category
- **Quick setup flow** — Streamlined 3-step installation process

## v1.10 Feature Recap

### 🚀 Smart Context Engine
- **Pattern Learning** — `PatternLearner` learns from task execution, predicts next actions
- **Auto-Summary** — `AutoSummary` extracts key changes from diffs automatically
- **Incremental Loading** — `IncrementalLoader` with LRU cache for skills and context

### 🤖 Multi-Agent Orchestration
- **Delegation Hints** — `DelegationHints` provides complexity scoring, agent recommendation, parallel candidates
- **Batch Dispatch** — `hive_worktree_batch` starts multiple tasks in parallel
- **Auto Agent Selection** — Recommends best agent based on task type

### ⚡ Performance
- **Background Sync** — `BackgroundSync` for non-blocking periodic sync
- **Lazy MCP Init** — Fast plugin startup with lazy initialization
