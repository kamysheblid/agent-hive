# v1.5.0 Release Notes

**Date**: 2026-03-20

## Highlights

This release adds **Tier 1 package integrations** (veil, dora, auto-cr), **LSP auto-install**, and **auto-dependency installation** for a more seamless experience.

## New Features

### @ushiradineth/veil MCP

Code discovery and intelligent retrieval MCP server.

```json
{
  "mcpServers": {
    "veil": {
      "command": "npx",
      "args": ["-y", "@ushiradineth/veil@latest", "mcp", "server"]
    }
  }
}
```

**Tools:**
- `veil_discover` - Get files, symbols, and code chunks in one step
- `veil_lookup` - Get most relevant context for task
- `veil_files`, `veil_symbols`, `veil_search` - Focused follow-up

### @butttons/dora CLI Tools

SCIP-based code navigation tools.

| Tool | Description |
|------|-------------|
| `dora_status` | Check installation/index status |
| `dora_symbol` | Find symbol definitions |
| `dora_file` | Get file dependencies |
| `dora_references` | Find symbol references |
| `dora_cycles` | Detect circular dependencies |
| `dora_unused` | Find unused/dead code |

### auto-cr-cmd Tools

SWC-based automated code review.

| Tool | Description |
|------|-------------|
| `auto_cr_status` | Check installation |
| `auto_cr_scan` | Scan directory for issues |
| `auto_cr_diff` | Scan git diff |
| `auto_cr_rules` | List available rules |

**Built-in Rules:**
- no-deep-relative-imports
- no-circular-dependencies
- no-swallowed-errors
- no-catastrophic-regex
- no-deep-clone-in-loop
- no-n2-array-lookup

### LSP Auto-Install

New LSP management tools with automatic installation and fallback support.

| Tool | Description |
|------|-------------|
| `lsp_status` | Check LSP server status |
| `lsp_install` | Install LSP for a language |

**Supported Languages:**
- TypeScript/JavaScript (typescript-language-server, volar)
- Python (pyright, ruff-lsp, jedi-language-server)
- Rust (rust-analyzer)
- Go (gopls)
- Java (eclipse-jdtls)
- C/C++ (clangd, ccls)
- C# (OmniSharp)
- Ruby (solargraph)
- PHP (phpactor)
- Vue (volar)
- Svelte (svelte-language-server)

### Auto-Dependency Installation

Plugin automatically installs required CLI tools on startup:
- `btca-cli` - Bluetooth Classic Audio
- `@butttons/dora` - Code navigation
- `auto-cr-cmd` - Code review
- `scip-typescript` - TypeScript indexer

Configure via `agent_hive.json`:
```json
{
  "autoInstallDeps": true
}
```

### Model Selector

Free model priority for simple tools to optimize costs.

Tools categorized by complexity:
- **Free**: status checks, diagnostics, simple operations
- **Standard**: normal coding operations
- **Premium**: complex refactoring, code generation
