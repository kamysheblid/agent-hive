# v1.6.10

## Summary

Fixed critical installation failure caused by `@sparkleideas/memory` package having a broken dependency on `@sparkleideas/agentdb@alpha`.

## Changes

### Bug Fixes
- **Moved `@sparkleideas/memory` to optionalDependencies** - The package has a broken dependency chain (`@sparkleideas/agentdb@alpha` doesn't exist)
- **Updated test** - Vector memory test now handles both fallback and vector modes

### Doctor Tool
- `@sparkleideas/memory` is still checked by doctor but is optional
- Doctor recommends installing it separately if needed

## Installation

```bash
npm install @hung319/opencode-hive@1.6.10
```

For Node.js v24+:
```bash
export CXXFLAGS="-std=c++20"
npm install @hung319/opencode-hive@1.6.10
```

## Upgrade Path

No breaking changes. Upgrade from v1.6.9 or earlier.
