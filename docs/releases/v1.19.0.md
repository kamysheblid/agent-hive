# v1.19.0

## Features

### Shell PATH Auto-Configuration
- **Auto-detect shell type** — bash, zsh, fish, sh (POSIX)
- **Auto-add hive bin to PATH** — writes `export PATH=".../hive/bin:$PATH"` to `~/.bashrc`, `~/.zshrc`, `~/.config/fish/config.fish`, or `~/.profile`
- **Idempotent** — skips if PATH already present in config
- **Clean bash output** — removes `PATH="..."` prefix from every bash command
- Runs once on plugin init, not per-command

### Changes
- New utility: `shell-path.ts` — shell config detection and PATH injection
- Removed PATH prefix from `tool.execute.before` hook — commands no longer show PATH
- PATH now inherited naturally from shell environment
