# Release v1.4.0

## New Features

### Memory System (Letta-style)
- **Memory Tools**: `hive_memory_list`, `hive_memory_set`, `hive_memory_replace`
- **Journal Tools**: `hive_journal_write`, `hive_journal_search`
- **System Prompt Injection**: Memory blocks automatically injected into system prompt
- **Default Blocks**: `persona`, `human`, `project` blocks auto-created on first run
- **Storage Locations**:
  - Global: `~/.config/opencode/hive/memory/` (cross-project)
  - Project: `.hive/memory/project/` (project-scoped)

## Context Optimization

### Token Truncation Hook
- Automatically truncates tool outputs >30k characters
- Keeps first 40% + last 40% of content
- Configurable via `tokenTruncation` in `agent_hive.json`

### Session Snapshot Hook
- Captures active feature and pending tasks before compaction
- Injects resume context after compaction
- Configurable via `sessionSnapshot` in `agent_hive.json`
