# v1.10.7

## Bug Fixes

- **Fixed detection of globally installed packages** — Check `/root/.local/node_modules` for packages installed via `--prefix`

### Root Cause

When running `npm install --prefix /root/.local`, packages are installed to `/root/.local/node_modules/` but doctor was only checking the local project via `npm list`.

### Fix

Now checks multiple locations:
- Current project `node_modules`
- `/root/.local/node_modules` (global prefix)
- `/usr/local/node_modules`

### Before (missed detection)
```
🚀 Agent Tools (0/3)
   ○ @sparkleideas/agent-booster not installed  # ❌ Actually installed!
```

### After (correct detection)
```
🚀 Agent Tools (3/3)
   ✅ @sparkleideas/agent-booster v0.2.3

   ✅ @ast-grep/napi v0.41.1
```
