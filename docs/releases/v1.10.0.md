# v1.10.0

## Summary

Multi-Agent Orchestration features for better task delegation and parallel execution.

## New Features

### 1. Delegation Hints

Worktree launch now includes intelligent hints:

```json
{
  "delegationHints": {
    "estimatedComplexity": "medium",
    "recommendedAgent": "forager-worker",
    "parallelCandidates": ["task-2", "task-3"],
    "estimatedTime": "30-60 min",
    "warnings": []
  }
}
```

**Complexity Levels:**
- `simple`: 5-15 min (fix typos, comments)
- `medium`: 30-60 min (features, refactors)
- `complex`: 1-3 hours (architectural changes)

### 2. Batch Task Dispatch

New tool: `hive_worktree_batch` for parallel task execution.

```typescript
// Start multiple independent tasks at once
hive_worktree_batch({
  tasks: ["task-1", "task-2", "task-3"],
  feature: "my-feature"
})
```

**Returns:**
```json
{
  "success": true,
  "batch": true,
  "total": 3,
  "succeeded": 3,
  "failed": 0
}
```

### 3. Agent Recommendations

Automatically suggests the best agent based on task type:

| Task Type | Recommended Agent |
|-----------|------------------|
| Simple | forager-worker |
| Medium | forager-worker |
| Complex | architect |
| Security | hygienic-reviewer |

## Changes

| Component | Enhancement |
|-----------|-------------|
| `hive_worktree_start` | Now includes delegation hints |
| `hive_worktree_batch` | NEW - Parallel task dispatch |
| Worktree launch | Complexity + time estimates |

## Installation

```bash
npm install @hung319/opencode-hive@1.10.0
```

## Usage

```bash
# Start single task with hints
hive_worktree_start({ task: "task-1" })

# Start multiple tasks in parallel
hive_worktree_batch({ tasks: ["task-1", "task-2", "task-3"] })
```
