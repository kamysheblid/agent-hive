# v1.15.0

## Features

### CRW MCP (Firecrawl Alternative)

- **New built-in MCP**: `crw-mcp` — web scraping & crawling, no backend needed
- **6 tools**: `crw_scrape` (URL → markdown), `crw_crawl` (BFS crawl), `crw_check_crawl_status`, `crw_map` (site discovery), `crw_search`, `crw_parse_file` (PDF)
- **Two modes**: Embedded (zero setup) or Proxy (`CRW_API_URL` + `CRW_API_KEY`)
- Scout agent updated with CRW tools in tool strategy
- Auto-activates when env vars are set

### Batch Edits Optimization

- **Forager worker prompt** updated with batch-edits guideline
- Prefer `write` full file when >30% changes, `hive_code_edit` for medium changes
- Batch reads: avoid tiny slices, read 200-500 lines at once
- Reduces round-trips, speeds up task execution

## Breaking Changes

- None (CRW is additive, batch edits are guidelines only)
