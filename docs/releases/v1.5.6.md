# v1.5.6

## What's Changed

### Re-add ast-grep native tools with proper native binary handling

Following [oh-my-openagent](https://github.com/code-yeongyu/oh-my-openagent)'s pattern, we've re-added `@ast-grep/napi` with proper configuration for Node.js v24 compatibility:

- **Added `@ast-grep/napi` as regular dependency** (not optional)
- **Externalized in build command** to avoid bundling native binaries
- **Added `bunfig.toml`** with `trustedDependencies` config
- **Added `BUN_INSTALL_ALLOW_SCRIPTS` env var** to CI workflows

### Tools restored

| Tool | Description |
|------|-------------|
| `ast_grep_dump_syntax_tree` | Dump code's syntax structure for debugging |
| `ast_grep_test_match_code_rule` | Test code against YAML rules |
| `ast_grep_find_code` | Find code matching patterns in project |
| `ast_grep_scan_code` | Scan for TypeScript bugs and best practices |
| `ast_grep_rewrite_code` | Transform/refactor code with AST patterns |
| `ast_grep_analyze_imports` | Analyze import usage in codebase |

### MCP restored

- `ast_grep`: Local npx-based ast-grep via `@notprolands/ast-grep-mcp`

## Bug Fixes

- Fixed potential build issues on Node.js v24 by properly configuring native binary postinstall scripts
