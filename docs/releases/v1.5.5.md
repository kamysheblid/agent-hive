# v1.5.5 Release Notes

**Date**: 2026-03-20

## Highlights

This release removes the ast_grep MCP (causes issues on some systems) and adds CLI tool detection to `hive_doctor` for better dependency management.

## Changes

### Removed

- **Removed `ast_grep` MCP** from built-in MCPs
  - Was using `@notprolands/ast-grep-mcp` via npx
  - Use native ast-grep tools directly or install separately if needed

### Added

- **CLI Tool Detection in hive_doctor**
  - Now checks for: dora, auto-cr, veil, scip-typescript
  - Shows which tools are available via npx
  - Provides install commands for missing tools

## Migration

### If You Need ast_grep

Install separately or use MCP-based version:

```json
// In your opencode.json MCP config
{
  "mcpServers": {
    "ast_grep": {
      "command": "npx",
      "args": ["-y", "@notprolands/ast-grep-mcp"]
    }
  }
}
```
