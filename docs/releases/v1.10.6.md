# v1.10.6

## Bug Fixes

- **Fixed hanging issue** — MCP servers (like searxng) were hanging because `npx` runs them as daemons

### Root Cause

When checking/installing CLI tools, running `npx -y mcp-searxng --version` actually starts the MCP server which waits for STDIO connections (blocking behavior).

### Fix

1. **Detection** — Added `2>&1 || true` and `stdio: 'pipe'` to prevent hanging on MCP servers that don't have `--version`
2. **Installation** — Use `npm install -g` instead of running CLI commands that may hang

### Before (hung)
```bash
$ npx -y mcp-searxng --version
🔍 MCP SearXNG Server v0.10.0 - Ready
✅  Configuration valid
🌐 SearXNG URL: https://search.rhscz.eu
📡 Waiting for MCP client connection via STDIO...
# (hangs forever)
```

### After (works)
```bash
$ npm install -g mcp-searxng
# Installs without running CLI
```
